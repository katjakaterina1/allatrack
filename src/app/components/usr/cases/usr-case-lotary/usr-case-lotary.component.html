Атрибуты фискального чека

Совокупность налогового номера, фискального номера, суммы и даты выдачи чека является уникальной комбинацией. Данная комбинация может служить в качестве идентификатора лотерейного билета. Так каждый чек станет билетом в лотерее.

Автоматическое распознавание на чеке

В связи с наличием опыта, можем предложить автоматизировать внесение данных из чека в базу. Предлагаем реализовать эт:о с помощью распознавания параметров чека по фото на мобильном телефоне.

Для распознавания будет использована самообучаемая система. Для этого потребуется:

Стандартизировать шрифты и расположение уникальных элементов чека.

Для обучения системы необходимо собрать базу фотографий чеков. Данные можно наработать в процессе «ручного» этапа акции.

Обучение системы займет 2 месяца после окончания сбора базы фотографий.
