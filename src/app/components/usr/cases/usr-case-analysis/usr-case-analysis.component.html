Техническое решение проблемы

Использование технологии Blockchain обезопасит пользователей благодаря прозрачности и защищенности, так как блокчейн это неизменный регистр, в который записываются все транзакции-записи между членами сети.

Децентрализованная база данных Blockchain решает проблему центрального сервера, который можно взломать. Кроме того, она работает на ассиметричном шифровании с длиной ключа 256 бит, являющемся в этом смысле супербезопасным паролем.

Генерируется уникальный токен по указанной при регистрации информации пользователя. Этот токен внедряется в базу данных Blockchain.
